## 流式端点：`/streaming/hidden_danger` — 纯文本响应

### 示例 1：隐患场景图 — 28 项（含重复）

**图片**：`0隐患案例.png` (1.9MB)
**耗时**：10.99s
**输出**：28 项纯文本（含法规依据），其中 4 项主题不同，24 项几乎重复

```
1.管道未固定，存在跌落风险。整改措施：固定管道。
隐患依据：《固定式升降工作平台》（JB/T 11169-2011）第5.5.2条。固定式升降工作平台的周边应有防止人员和材料跌落的保护装置...

2.地面上有散落的物品和杂物，可能会造成滑倒或绊倒。整改措施：清扫地面，确保无杂物和易滑物品，确保地面整洁。
隐患依据：《危险化学品企业特殊作业安全规范》（GB 30871-2022）第8.2.7条。易滑动、易滚动的工具、材料堆放在脚手架上时，应采取防坠落措施。

3.地面上有积水，可能会导致滑倒事故。整改措施：清理积水，确保地面干燥，或者设置警示标志提醒工人注意安全。
隐患依据：《起重机械安全规程 第5部分：桥式和门式起重机》（GB/T 6067.5-2014）第3.4.1条；第3.4.2条。室外工作的桥式和门式起重机，通道与平台上应采取防积水措施...
```

**⚠️ 重复项特征**：第4~28项几乎完全一致（"地面上有散落的工具和材料，可能会造成绊倒或伤害。整改措施：将所有工具和材料整齐地放置在指定的位置"），仅序号不同。输出时需去重。

### 示例 2：地面裂缝

**图片**：微信图片_20260616140401_365_3496.jpg (9.2MB)
**耗时**：11.53s
**输出**：1 项

```
1.地面上有裂缝和破损，可能会导致人滑倒或物品掉落。整改措施：需要对地面进行修补和加固，确保其平整和稳固。
隐患依据：《低温仓储作业规范》（GB/T 31078-2024）第7.5.4条。不得将物品直接接触地面及墙面。
```

## 响应格式规律

| 字段 | 纯文本响应 |
|------|------------|
| 序号 | `1.` `2.` ... |
| 隐患描述 | `<序号>.<描述>` |
| 整改建议 | `整改措施：<建议>` |
| 法规依据 | `隐患依据：<标准>`（约80%项有） |
| 无隐患标记 | 纯文本 `图片中没有安全隐患。` |
