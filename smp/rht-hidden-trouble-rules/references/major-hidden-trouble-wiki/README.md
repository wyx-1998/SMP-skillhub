# 重大隐患判定知识库

本目录是隐患评估的重大隐患基础知识层，由条款拆解结果编译而成，用于选择适用标准、定位候选规则、理解判定逻辑和识别待补要件。

## 使用边界

- 先确认行业、对象和场景，再进入对应标准。
- 先读标准总览、适用范围和判定总则，再读专题或条款页。
- 本知识库是辅助解释和路由资料，不单独证明法规现行有效，也不替代法规原文。
- 判定重大隐患前，必须核对完整构成要件、逻辑关系、数量或程度门槛及排除情形。
- 未命中重大隐患规则，不等于不构成隐患或风险较低。
- 页面中标记的后续编译建议、待补内容和草稿状态不得当作正式判定依据。
- 专题页中的 `条款拆解/` 路径是来源仓库的溯源标识，本目录不包含对应中间文件；需要核验原文时应回到正式法规、标准或外部法规知识库。
- 页面保留法规和标准中的原文术语。使用时只提取与当前登记隐患相关的适用范围、构成要件、门槛和排除情形。

## 推荐检索方式

先限制到适用标准目录，再使用对象、异常状态、控制措施和阈值组合检索：

```bash
rg -n -S "对象词|异常状态|控制措施|阈值词" <适用标准目录>
```

例如：

```bash
rg -n -S "有限空间|通风|检测|监护" 工贸企业重大事故隐患判定标准
rg -n -S "报警|联锁|失效|无效" AQ\ 3067-2026化工和危险化学品生产经营企业重大生产安全事故隐患判定标准
```

检索到专题页后，回到同目录的总览、范围或判定总则核对适用性，不以单个关键词命中直接定性。
